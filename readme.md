# PDF Annotation Tool

A web-based PDF Annotation Tool that allows users to upload, view, and annotate PDF documents seamlessly. Leveraging modern web technologies, this tool provides an intuitive interface for managing PDF annotations, enhancing document review and collaboration processes.
